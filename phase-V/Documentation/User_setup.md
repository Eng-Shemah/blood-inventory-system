## Quick Start
1. Execute `01_database_creation.sql` as SYSDBA
2. Run `02_tablespaces.sql` in the new PDB as blood_admin
3. Execute `03_user_setup.sql` to create application user
4. Run `04_configuration.sql` for system configuration
5. Execute `05_verification_queries.sql` to verify setup
